1 - deve ser executado dentro de um repositório git. Vai listar o id de todos os commits no repositório em ordem crescente. Lista completa de informações a serem obtidas a partir da flag --pretty: https://git-scm.com/book/en/v2/Git-Basics-Viewing-the-Commit-History

git log --first-parent --pretty="%H" > ../../github_commits/_commits.txt



2 - comando para configurar um repositório git de acordo com um certo commit. A opção -C só é utilizada quando o comando é executado de um diretório fora do repositório

git -C <caminho_para_repositorio> reset  --hard <commit_id>
